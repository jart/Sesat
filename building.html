Building Sesat from source

Check out the latest version of Sesat from the Git repository:

svn co https://michaelsembwever.github.io/Sesat/svn/sesat-kernel/trunk sesat-kernel

From the working copy of sesat-kernel run the Maven as follows to build the artifacts, make sure you have installed all the requirements:

mvn install
